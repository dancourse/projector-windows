## Quick Start Guide

### Step 1: Review the Research
Read through **brand-research.md** to understand:
- Which programs are easiest to join (Anker, XGIMI, Nebula)
- Which require formal applications (BenQ, Epson)
- Commission rates and program restrictions
- Recent brand initiatives and positioning

### Step 2: Customize Templates
Use **pitch-template.md** to prepare your outreach emails:
1. Choose the appropriate template for each brand
2. Personalize with specific details:
   - Your role/title
   - Projector Windows website/app link
   - Any relevant metrics (user base, engagement)
   - References to recent brand launches
3. Save customized versions or send directly

### Step 3: Apply to Programs
Priority order:
1. **Week 1**: Anker, XGIMI, Nebula (quick affiliate sign-ups)
2. **Week 2**: BenQ (formal application)
3. **Week 3**: LinkedIn outreach to all brands
4. **Month 2+**: Epson (after proving concept)

### Step 4: Track Progress
Open **outreach-tracker.html** in your browser:
- Update status badges as you make contact
- Log responses and feedback in brand sections
- Track acceptance rates and commission performance
- Monitor follow-up dates

## Key Messaging

### For All Brands
**Core Value Proposition:**
"Projector Windows is a macOS app that enhances the projector experience by solving window management pain points. We want to recommend [Brand] projectors to our users and earn commission on referrals while providing better software support for your Mac customers."

### Brand-Specific Angles
- **Nebula**: Portable projector use cases, Google TV integration
- **XGIMI**: 4K/laser capabilities, professional-grade positioning
- **Anker**: Ecosystem approach (Anker + Nebula + Projector Windows)
- **BenQ**: Gaming/business/education segments, technical partnership
- **Epson**: Enterprise value, AV consulting, Lifestudio + Gemini AI

### Use Case Examples
1. **Presentation Mode**: One-click optimized layout for slides + notes + video
2. **Movie Night**: Saved configuration for full-screen on projector, controls on laptop
3. **Gaming Setup**: Optimized windows for gaming on projector with secondary displays
4. **Teaching**: Quick profile switching between lecture, demo, collaboration modes
